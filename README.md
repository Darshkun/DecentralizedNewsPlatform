## 📌 Project Description
Decentralized News Platform is a blockchain-based application built on Stellar Soroban that allows users to publish and access news without relying on centralized authorities. The goal is to promote transparency, censorship resistance, and trust in news distribution.
<img width="959" height="506" alt="image" src="https://github.com/user-attachments/assets/90b46e76-5e94-40a5-980b-89247c59ae97" />

## 🚀 What it does
This smart contract enables users to:
- Publish news articles on-chain
- Store content in a decentralized manner
- Retrieve published news using unique IDs

Each news article is permanently recorded on the blockchain along with the author's address, ensuring authenticity and immutability.

## ✨ Features
- 🔐 **Decentralized Publishing** – No central authority controls the content
- 👤 **Author Verification** – Only authenticated users can publish news
- 🧾 **Immutable Records** – Once published, news cannot be altered
- 🔍 **Easy Retrieval** – Access news using unique IDs
- ⚡ **Lightweight Contract** – Optimized for Soroban environment

## 🛠 Tech Stack
- **Stellar Soroban**
- **Rust (Smart Contract)**
- **Soroban SDK**

## 📦 Smart Contract Functions
- `publish_news(author, content)` → Publish a news article
- `get_news(news_id)` → Retrieve a news article
- `total_news()` → Get total number of published articles

## 🌍 Use Cases
- Independent journalism
- Censorship-resistant media platforms
- Community-driven news systems
- On-chain content verification
